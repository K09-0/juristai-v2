## 🎯 Основные функции

### 1️⃣ Поиск по законодательству РК

1. Нажмите на кнопку **"Задать вопрос"**
2. Введите ваш вопрос о законодательстве РК
3. Система найдет релевантные статьи и даст ответ
4. Каждый ответ содержит ссылку на источник

**Примеры вопросов:**
- "Какие сроки исковой давности в РК?"
- "Что такое трудовой договор?"
- "Какие права у потребителя при покупке товара?"

### 2️⃣ Генерация документов

1. Перейдите в раздел **"Создать документ"**
2. Выберите тип документа:
   - **Исковое заявление** - для подачи в суд
   - **Претензия** - для отправки контрагенту
   - **Договор** - различные виды договоров
3. Заполните необходимые поля
4. Выберите стиль (формальный, нейтральный, агрессивный, защитный)
5. Нажмите **"Создать"**
6. Экспортируйте в PDF или DOCX

### 3️⃣ Мониторинг поправок

1. Перейдите в раздел **"Мониторинг"**
2. Выберите интересующие вас законы
3. Система будет отправлять уведомления о новых поправках
4. Сравнивайте старую и новую версию статей

## 🐛 Проблемы и решения

### Сайт не загружается
1. Очистить кэш браузера (Ctrl+Shift+Delete)
2. Перезагрузить страницу (F5)
3. Попробовать в другом браузере

### Ошибка при входе
1. Проверить правильность логина и пароля
2. Убедиться, что caps lock выключен
3. Попробовать волшебную ссылку (email magic link)

### Поиск не работает
1. Проверить интернет соединение
2. Убедиться, что вы авторизованы
3. Попробовать более простой вопрос

# JuristAI v2 - Дизайн-концепции

## Выбранная концепция: Минималистичный Функционализм

### Design Movement
**Современный минимализм с элементами функционального дизайна** — вдохновлено Европейским минимализмом (Swiss design) и современными LegalTech платформами (Notion, Linear, Figma). Фокус на ясности, функциональности и доступности.

### Core Principles
1. **Максимальная ясность** — каждый элемент имеет назначение, ничего лишнего
2. **Иерархия через пространство** — используем отступы и размер вместо цвета для создания структуры
3. **Контрастная типография** — смешиваем шрифты для визуального разделения (заголовки vs тело)
4. **Адаптивная тема** — плавное переключение между светлой и тёмной темой без потери читаемости

### Color Philosophy
- **Светлая тема**: Нейтральный фон (белый/серый), тёмный текст, синий акцент для действий
- **Тёмная тема**: Глубокий тёмный фон (slate-900), светлый текст, яркий синий для контраста
- **Акцентный цвет**: Профессиональный синий (blue-600/700) для кнопок и интерактивных элементов
- **Вторичный цвет**: Серый для вспомогательного текста и разделителей

### Layout Paradigm
- **Асимметричная сетка** — главная колонна контента (2/3) + боковая панель (1/3) для фильтров/навигации
- **Вертикальный ритм** — строгие интервалы между элементами (8px, 16px, 24px, 32px)
- **Максимальная ширина контента** — 1200px для оптимальной читаемости

### Signature Elements
1. **Карточки с мягкой тенью** — тонкие тени для глубины без визуального шума
2. **Скруглённые углы (8px)** — не слишком округлые, сохраняют профессионализм
3. **Иконки Lucide** — минималистичные, чистые линии, идеально подходят для юридического контекста

### Interaction Philosophy
- **Микро-взаимодействия** — плавные переходы (200-300ms) при наведении и клике
- **Обратная связь** — тосты для уведомлений, скелетоны для загрузки
- **Клавиатурная навигация** — полная поддержка Tab, Enter, Escape

### Animation
- **Входящие элементы** — fade-in + slight-up (200ms ease-out)
- **Наведение на кнопки** — scale(1.02) + shadow increase
- **Переходы между темами** — плавный fade (300ms)
- **Загрузка** — спиннер с мягким вращением

### Typography System
- **Заголовки**: Geist Sans (Google Fonts) — bold, 600-700 weight
  - H1: 36px/1.2 (hero)
  - H2: 28px/1.3 (section)
  - H3: 20px/1.4 (subsection)
- **Тело текста**: Inter (Google Fonts) — regular 400, 16px/1.6
- **Код/моноширинный**: JetBrains Mono — для примеров законов и цитат

---

## Реализация
Выбран **Минималистичный Функционализм** для создания профессионального, чистого интерфейса, который:
- Не отвлекает от контента (законов и ответов AI)
- Обеспечивает быструю навигацию для юристов
- Поддерживает обе темы без потери функциональности
- Масштабируется от мобильных до десктопных устройств
